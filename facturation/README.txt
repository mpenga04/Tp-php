SYSTEME DE FACTURATION - PHP PROCEDURAL

1) PREREQUIS
- XAMPP (Apache + PHP)
- Placer le dossier "facturation" dans htdocs/Tp php/

2) LANCEMENT
- Demarrer Apache dans XAMPP
- Ouvrir: http://localhost/Tp%20php/facturation/auth/login.php

3) COMPTE PAR DEFAUT
- Identifiant: admin.root
- Mot de passe: Admin@123

4) ROLES
- super_admin: gestion comptes + manager + caissier
- manager: produits + rapports + facturation
- caissier: facturation + lecture produits
